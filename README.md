Local install
-------------

Setup and activate a python3 virtualenv via your preferred method. e.g. and install production requirements:

    $ make ve

To remove virtualenv:

    $ make clean

To install github hooks:

    $ make install_hooks

Create .env file and fill the data:

    $ cp .env.example .env

Export env variables:

    $ export $(xargs < .env)

### Env variables description:

Variable for enable testing:

* `TELEGRAM_BOT_TOKEN` - Telegram bot token generated by BotFather for access to bot chat.
* `TELEGRAM_CHAT_ID` - Telegram chat ID where bot will send message.
* `HURMA_HOST` - Base URL where Hurma service was deployed.
* `HURMA_EMAIL` - Email from Hurma service.
* `HURMA_PASSWORD` - Password from Hurma service.

Run BOT:
-------------
For run bot locally:

    $ python main.py
